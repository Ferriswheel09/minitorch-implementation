## Setup:
First, ensure that venv is set up via: python -m venv .venv
1. Run source .venv/Scripts/activate
2. Run: 
```
python -m pip install -r requirements.txt
python -m pip install -r requirements.extra.txt
python -m pip install -Ue .
```
